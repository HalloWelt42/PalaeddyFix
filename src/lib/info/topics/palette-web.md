Der Begriff „Webfarbe" umfasst zwei historisch gewachsene Paletten-Systeme:

## CSS Named Colors

148 Farben mit festen Namen wie `red`, `dodgerblue`, `rebeccapurple`, die direkt per Name in CSS verwendet werden können. Die Liste stammt aus den frühen Browser-Tagen (X11-Farbnamen wurden übernommen), wurde 2014 um `rebeccapurple` ergänzt und ist seitdem unverändert.

**Vorteil:** kurze, lesbare Notation statt `#87ceeb` schreibt man einfach `skyblue`.

## Websichere Paletten (216-Farben)

Die 216 Farben, die sich aus allen Kombinationen von `00`, `33`, `66`, `99`, `cc`, `ff` pro Kanal ergeben. Ursprünglich als „Netscape Color Cube" bekannt: auf Monitoren mit 256 gleichzeitig darstellbaren Farben konnten diese 216 ohne Dithering angezeigt werden.

## Historischer Kontext

Die Web-Safe-Palette hat heute kaum mehr praktische Bedeutung -- alle modernen Displays können Millionen Farben. Sie ist aber ein schöner Baukasten für Retro-Designs, Pixel-Art und systematische Test-Paletten.

## Typischer Einsatz

- CSS Named Colors: als praktischer Shortcut in CSS, besonders für schnelle Prototypen
- Web-Safe: als Palette für Pixel-Art oder nostalgische Designs
