Tailwind CSS ist ein Utility-First-CSS-Framework, das 2017 von Adam Wathan vorgestellt wurde. Seine Farbpalette ist heute eines der meistkopierten Paletten-Systeme im Web: 22 Farbfamilien (Slate, Gray, Zinc, Neutral, Stone, Red, Orange, Amber, Yellow, Lime, Green, Emerald, Teal, Cyan, Sky, Blue, Indigo, Violet, Purple, Fuchsia, Pink, Rose) mit je 11 Shades von 50 (sehr hell) bis 950 (sehr dunkel).

## Aufbau

- **Shade 50**: Hintergrund, fast weiß.
- **Shade 500**: Standard-Akzentfarbe, mittlere Helligkeit.
- **Shade 900/950**: Text oder sehr dunkler Hintergrund.
- Alle Schritte sind perzeptiv nahezu linear gestaffelt -- gut für gleichmäßige Kontraste.

## Warum bekannt

Weil das System sehr konsistent und vorhersagbar ist: Jeder Tailwind-Shade sieht in jeder Farbfamilie ähnlich hell/dunkel aus. Das erleichtert Theme-Wechsel und Design-Systeme erheblich.

## Typischer Einsatz

Als fertige Design-System-Grundlage für Web-Projekte, als Referenzpalette für Farbharmonie oder als Match-Palette in PaläddyFix, wenn man eine Bildfarbe auf die nächstgelegene Tailwind-Farbe abbilden will.
